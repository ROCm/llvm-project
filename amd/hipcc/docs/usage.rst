******************************************
Using HIPCC
******************************************

To use the newly built the ``hipcc`` and ``hipconfig`` executables from the build folder use ``./`` in front of the executable name.
For example:

.. code-block:: shell

    ./hipconfig --help
    ./hipcc --help
    ./hipcc --version
    ./hipconfig --full
